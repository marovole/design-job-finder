# Design Project Finder Output

This directory contains results from the design job finder skill.

## Files

### Data Files
- `design_projects_YYYY-MM-DD.csv` - Complete project data with all fields
- `contact_list_YYYY-MM-DD.csv` - Filtered list with only contact information
- `design_projects_summary.md` - Statistical summary and recommendations

### Marketing Emails
- `marketing_emails/high_priority/` - Emails for A/B级 projects
- `marketing_emails/medium_priority/` - Emails for C级 projects

## Usage

### Reviewing Projects
1. Open `design_projects_summary.md` for a quick overview
2. Check `design_projects_YYYY-MM-DD.csv` for full details
3. Filter by priority or platform as needed

### Contact Outreach
1. Open `contact_list_YYYY-MM-DD.csv` for easy copy/paste
2. Review emails in `marketing_emails/` folder
3. Personalize before sending
4. Track responses in your CRM

## Recommendations

- Start with A/B级 projects (highest priority)
- Personalize the first 5-10 emails before bulk sending
- Send 10-15 emails per day to avoid spam filters
- Follow up after 5-7 days if no response
